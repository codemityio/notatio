## Summary

A tool to convert `mmd` files to `svg`/`png` images.

## Manual

``` bash
$ notatio mermaid --help
NAME:
   notatio mermaid

USAGE:
   notatio mermaid [command options]

DESCRIPTION:
   A tool to convert mmd files to svg/png images.

OPTIONS:
   --input-path value                  input path, either a file to be converted or a directory to be scanned (default: ".")
   --output-format value               output format (svg or png) (default: "svg")
   --puppeteer-config-json-path value  puppeteer-config.json file path (default: "/usr/local/lib/puppeteer-config.json")
   --recursive                         enable recursive directories scan (default: false)
   --help, -h                          show help
```
